javascript 脚本语言
js主要运用浏览器处理用户输入实现交互
比如:注册用户，js作用就是把注册信息发送到服务器，也可以校验用户输入信息是否正确

1.准备
a.开发工具
	vscode
	sublime text
b.js基础
	b1.写入js代码的方式
		1>直接写入body中
		2>直接写入html标签中
		注意:常用的信息输出方式是用alert()方法以消息框形式输出信息，也可以通过document.write()写入到网页中
	b2.浏览器对js的支持	
		navigator.appName      浏览器的名称
		navigator.appVersion   浏览器版本号
		navigator.appCodeName  浏览器内部发行代号
	b3.注意事项
		1>大小写敏感
		2>空格和换行
			代码中多余的空格会被忽略，同一个标识符所有字母必须连续，一行代码可以分成多行来写
		3>分号
			作为一个语句结束的标志
			分号之后是新语句的开始
			当一行只有一个程序语句时候，可以使用分号
c.js的数据类型
	c1.字符类型
		c11.字符操作基础
		字符类型是通过引号括起来的文本
			"55开说,我没开挂！！"
			'我很帅'
		定义字符串的方式
		通过双引号
			var hello="你瞅啥？"
		通过单引号
			var hello='瞅你咋地？'
		c12.特殊符号
		\n     回车
		\t     缩进
		\r     换行，相当于回车
		\'     单引号
		\"     双引号
		注意:\也叫转义符，显示特殊符号时候使用
	c2.数值型数据
		数值类型数据没有分整型、浮点、双浮点等等，可以使用通用方法和科学计数法
		js中数据可以写成十进制、八进制、十六进制
		1>十进制可以使用普通记法和科学计算表示
			10;   数字
			10.1;  数字
			0.1;   数字
			3e7;   科学计数法
			0.3e7  科学计数法
		2>十六进制以0x或者0X开头后面跟0——F十六进制数据，没有小数或者指数
			0xAF3F
			0X30FB
		3>八进制以0开头，后面跟0-7的8进制数据，没有小数或者指数
			0123
			055
		注意:进制转换方式ParseInt、toString
	c3.布尔类型数据
		只有真或者假两个值得数据类型，作为逻辑表达式结果值真用true表示，假用false表示
		注意:非0就是真，0就是假
		定义布尔类型数据
			var b=true;
			var a=false;
	c4.复合数据类型
		数值、字符、布尔类型都是js的简单数据类型，复合数据类型有对象、数组等等
		1>常用的内置对象
		面向对象设计模式中，将数据和处理数据的方法捆绑在一块，形成一个整体成为对象
		对象封装数据和从操作数据的方法，雅士勇数据或者方法必须先创建一个对象，使用new来调用对象从而创建一个对象
		var obj=new object()  创建新对象
			Array        	数组对象，封装数组操作和属性
			Boolean 		布尔对象
			Date            日期对象，封装和日期相关操作和属性
			Maths           数学对象，提供基本的数学函数和常量
			Global          全局对象，全局函数和全局常量为该对象所有
		2>日期对象
		日期对象一个典型应用就是获取当前时间
		http://www.w3school.com.cn/js/jsref_obj_date.asp
		3>全局对象
		全局方法的拥有者
			isNaN()    		判断参数是否是非数字值，返回的是布尔值
			如果参数是特殊非数字值NaN,返回True
			parseInt()  	返回字符串转换得到的整数
			parseFloat() 	返回字符串转换得到的浮点数
			注意：NaN  not a number
		3>数学对象
		封装了和数学相关热性，包括一些常数和数学函数
		调用Math对象方法
		math.[{属性(方法名)}]
			PI    	返回圆周率
			abs     返回数字的绝对值
			sin   	返回数的正弦
			min  	返回给定数字中最小值
			max     返回给定数字中最大值
